# 2026-06-06

> Project: FieldQuanta/predict_molecule · Tags: #eval-result #decision

- 追到了声子全虚频的真凶, 不是之前以为的"模型学不会曲率"那么简单, 是个对称性 bug。 模型用了原子在晶胞里的绝对位置当特征, 导致整体平移一下晶体能量会变 —— 但物理上平移整个晶体能量必须不变 (空间是均匀的)。 这个性质一破, 声学振动模式 (最低频那几支) 数学上必然算成虚的, 怎么训都救不回。 根子是这套居中代码从分子项目继承来, 只居中了笛卡尔坐标, 没居中分数坐标。 修好后实测: 之前 KBr 的声学支是 **-7.7** (该是 0), 修完归零, 平移不变恢复。

- 但修完声子还是全虚频。 因为这个对称 bug 只破坏了"整体平移"那一类振动 (布里渊区中心), 修好它声学支对了, 其它振动模式 (区内部) 还是虚的 —— 那是能量面曲率算不准, 跟对称是两个独立的病。 所以"曲率是瓶颈"这个判断最终还是成立, 之前被对称 bug 盖住了。 曲率这个得靠更多更多样的数据或更大模型, 不是改架构能解决。

- 跑出了第一个真实的打榜 F1 (材料稳定性预测的主指标)。 我们 **0.638**, 同量级的 eSEN-3.2M 是 **0.568**, 我们更高; 能量预测误差也比它小。 但声子那条线大幅落后 (热导率分数 2.0 vs eSEN 0.878)。 一句话: 一阶性质 (稳不稳、能量) 超同级对手, 二阶性质 (振动) 差一截。 离北极星 0.9 还远, 那需要 OMat24 大数据预训练。 注意我们参数量是它 4.5 倍, 公平比要看算力。

- 之前加的"逐层迭代描述子"(让模型视野随深度变大) 测下来没用: 标准指标和声子都跟不加一样, 关掉了。

- 分析了误差最大的 50 个结构, 一个很清晰的规律: 全都是模型把 DFT 判定不稳的高能怪结构预测成了稳定 (假阳性)。 集中在氮化物、稀土 (f 区)、磁性过渡金属、部分氧化物, 都是训练数据 MPtrj 里少见的高能体系, 模型外推时过度自信。 这跟声子曲率差其实是同一个根: 数据覆盖不够。 两个病的解药都指向 OMat24 更大更多样的数据。

- 修完第一个对称 bug 声子还是全虚, 觉得"一个材料都算不出"太反常, 又往下挖, 结果挖出第二个 bug, 还是跟绝对坐标有关, 但这次在推理代码里不在模型。 先做了个关键对照: 拿一个已知正确的简单经验势 (金属的) 走我们完全一样的声子流程, 算出来全对 —— 证明流程没问题。 然后细查发现能量在原子平衡位置有个突跳 (力瞬间跳 9, 该连续), 把声子的二阶导算花了。

- 定位到突跳来自推理框架会把跑出晶胞的原子折叠回胞内, 原子越过胞边界时位置瞬间跳一个晶格长度。 我们模型按质心居中, 一个原子跳质心就跟着移一点, 所有原子的坐标特征都抖一下, 就这么制造了突跳。 关掉这个折叠 (我们模型本来就不需要它), 突跳从 9 降到 **0.005**, 能量光滑了。 一个澄清 (讨论中明确): 这不是用绝对坐标的错也不是质心系的错, 分子模型也这么干没事, 唯一祸首是折叠破坏了坐标系一致性。

- 但两个 bug 都修完声子还是全虚, 只是改善了 (某材料虚模式从 **45%** 降到 **22%**)。 多数振动模式算对了, 差几个软模没压到 0 以上, 而判据很严一个不达标就算虚。 觉得还是不对劲, 就去查竞品 eSEN 声子为什么那么好, 顺带看其它非等变模型怎么做的, 读了三篇论文。

- 查到一个关键技巧: 算声子的二阶导时用双侧差分 (正负都测取平均), 对非等变模型尤其重要 —— 因为我们模型不是严格对称, 对称等价的两个方向小有不一致, 双侧平均能抵消掉。 literature 说能把误差砍一半以上。 我们之前用的是单侧 (软件默认)。 改成双侧后**虚频幅度砍 3 到 10 倍**, 一批勉强虚的边缘材料就过线了。

- 过线之后又卡在算热导率那步报错, 查出来是声子软件 (phono3py) 大版本升级后接口改了名, 我们这套代码是老版本写的。 修了几处接口 (函数名、参数名、结果属性名), 热导率终于算出来了。

- 结果: 从**一个材料都算不出**, 到 **36/103 出真实热导率**, 热导率分数从算不出的 2.0 地板降到 **1.75**。 个别材料已经到 eSEN 水平 (误差 0.065-0.16)。 说明模型**能**做对一部分, 只是不一致 —— 大部分还是虚频或偏差大 (有的高估 2 倍)。 今天一共修了 4 个声子相关的 bug (两个模型对称性 + 双侧差分 + 软件接口)。 剩下的 1.75 是真曲率精度, 要靠减少径向基函数数量 (eSEN 用 10 个我们 64 个)、更大模型、OMat24 数据。 一阶 F1 能打、二阶声子从零到能出数, 是今天最大进展。
