# 2026-06-15

> Project: FieldQuanta/predict_molecule · Tags: #result #milestone

（背景:预测材料稳不稳定,F1 打分满分 1、目标 0.9。两条路线:"对称性硬编进网络"(等变)和"让网络从数据自己学"(非等变,我们走的)。这几天在搭一个非等变的"边通道"新架构。)

- **今天最大的结果:非等变边通道在大数据(OMat)全流程上,正式超过了我们之前最强的不变模型。** 完整 F1:**v3sh 0.7863 / v1 0.7803**,都赢过不变生产线的 **0.7775**。之前(6-12)在小数据快筛上就赢过(0.6855 vs 0.6434),今天在真正的大数据全流程(预训练→微调→弛豫)上**也赢了** —— **两条线都站住,这条非等变路子坐实了。**

- **最戏剧的是那个"寄予厚望却小数据垫底"的变体翻盘了。** v3sh(给边加了更丰富的角度特征,本来是冲着补"力"的短板去的)在小数据上力做得**最差**、F1 垫底;但在大数据上力做到**最好**、F1 也**最高(0.7863)**。完美印证了我们一条铁律:**加表达力的改动,小数据会骗你,只有大数据说真话。** 当时判它"被小数据委屈了、该上大数据",赌对了。

- 实现侧补了一串:把边通道做成 4 个可消融的变体(加性 / 双线性 / 角度特征 / 修正版双线性),加了一个**物理斥力项**(防止弛豫时原子撞穿、崩坏结构),并把架构补到**超过参考模型 ORB** 的程度。中途修好了前两天那个"乘法版数值爆炸"的隐患(加归一化压住),又顺手把一个评测脚本对齐了基线、好做公平对比。

- 离里程碑还有距离但方向很硬:现在 **0.786,离同级 eSEN-30M 的 0.819 还差 0.03,离北极星 0.9 还远**,但这是非等变这条路**第一次在大数据上真·超过不变线**,而且手里还有好几个没用的杠杆(修正版双线性 / ORB 节点更新 / 斥力项 / 放大规模)。下一步把这些杠杆一个个上,往 0.819 顶。

- 试了第一个杠杆(物理斥力项 ZBL),结果是**负的、弃用**。它在预训练阶段确实把能量做准了 ~3%(排斥墙交给物理公式、网络省容量),但**最终稳定性分数 F1 反而掉了**(0.7863 → 0.7759,甚至低于不变线 0.7775)。又一次印证"中间指标 ≠ 最终指标":它帮的是训练集内的能量,但对评测集(没见过的结构)的形成能反而略差;而且它本来要防的"弛豫时结构崩坏"在我们这压根没发生过(弛豫零失败)→ 没修到真问题、只添了扰动。**记一课:别被 in-domain 的提升骗了,F1 才算数。** 后面 v4 / 节点MLP 的消融不再叠 ZBL。
