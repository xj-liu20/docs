# 2026-06-23

> Project: FieldQuanta/predict_molecule · Tags: #result #experiment

- 开始做 CPS(Matbench 那个综合分 = F1 稳定性 + κ_SRME 热导 + RMSD 几何三块加权)。查下来:F1 我们有、κ_SRME 管线能跑,但 matbench_discovery 这个 pip 包卡在 1.3.1 没 CPS(CPS 只在它 GitHub 仓库、没发 PyPI),而且装 GitHub 版会改 API 把我们现有 F1/κ 脚本搞崩。所以完整 CPS 得单独建个环境 + 适配脚本,当独立小项目,先没动工作环境。

- 跑 6ep 模型的 κ_SRME(热导),smoke 两个材料全出虚频(imaginary freqs,算不出热导)。一度怀疑是不变性的 bug,但实测排除了。重点说旋转这块,因为非等变靠增广学不变性,本来以为只是近似:抽了 25 个不同成分/大小的 WBM 结构、每个先 0.05Å 微扰打破对称、再整体随机旋转 pos+cell,测 ΔE。结果 ΔE/atom 才 mean 0.0098 / max 0.046 meV,力协变误差 ~0.0003 eV/Å(对比单原子扰动 5.6 meV、力 MAE 0.05,小两三个量级)。而且 config 里 canonicalization 开关全是 false(没有 polar-frame、没有 sh-attn-gram),纯增广学出来的,跨结构稳健、不是偶然。所以非等变在不变性上不吃亏,推翻了之前的担心。

- κ_SRME 全 103 跑完了,结果反转:**61/103 出热导率,κ_SRME=1.5371**,根本不是 smoke 时以为的全虚频。中途看了一眼 ~45/103 全虚频就判"100% 虚频是 bug",其实是 partial-pool 假象,前段材料恰好虚频偏多,跟之前 F1 半程不可信一个坑。s20 那个老 run 也一样,实际 10/20 出 κ 不是 0。valid 的 κ 在 0.24–11.79 W/mK 物理合理区间。
- 而且这是我们目前最好的声子结果:c1(14M MPtrj)0/103 SRME 2.0 → smoothPS c1 36/103 SRME 1.75 → 现在 OAM 6ep 61/103 SRME 1.5371。说明 OMat+Alex+MPtrj 大数据预训练确实把曲率救回来不少,印证之前"更大模型/更多数据"那条 lever。剩 42 个还虚频,降虚频率仍是后面的抓手。离 SoTA(eSEN-OMat 0.17)还很远,但非等变能出热导率这件事坐实了。
