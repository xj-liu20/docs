### 3. base/finetune 拆解 + 训练预算纠偏

- **OMat-only 探针** (ref-adapted F1,干净对干净):我们 base **0.747** vs ORB OMat-only **0.7996**,弱 ~0.05;finetune 贡献 +0.04。瓶颈定位到 base,无泄漏水分。
- eSEN/ORB 配方 = **重 pretrain + 轻 finetune** (eSEN-30M-OAM = OMat 4ep pretrain + 1ep finetune);我们反了 (2ep pretrain + 4~8ep finetune)。OMat 收敛曲线证明 pretrain 没训透 (epoch2 仍降 E_pa -32%,末段 -6%/半 epoch)。
- → **战略:把预算从 finetune 挪到 pretrain。**
- **✅ 实证 (06-17): nmlpsh OMat pretrain 2ep→4ep → unique F1 0.8003→0.8151 (+0.0148) / full 0.7882→0.8031**。离 eSEN-30M-MPtrj 0.819 仅 ~0.004。**关键认知: finetune val 能量只动 -1.4% 但 F1 +0.0148 → 重 pretrain 经"力/弛豫质量 + OOD base 可迁移性"涨 F1,不靠 in-domain 能量**(F1=OOD 弛豫后 hull 分类,非静态能量 MAE)。重 pretrain 升级为 F1 主杠杆 → 续推 6ep/8ep。

### 4. LLPR 不确定度 + F1 误差诊断 (06-17)

实现了 LLPR (Last-Layer Prediction Rigidity):后处理、不重训,用末层特征协方差给每个预测一个绝对 σ。在 4ep 模型 + WBM 上的诊断:

- **σ 作能量不确定度成立**:高 σ 档实际 |误差| 0.057 > 低 σ 档 0.008 eV/atom(单调)。
- **σ 单独不预测 F1 判错**(判对 σ̄ 0.0462 ≈ 判错 0.0461):判错是**凸包边界效应**(余量 = |预测 each_hull| 小才翻),与误差大小正交。**σ + 余量组合才有用**:`|ep|<σ` 组判错率 18.6% vs 安全组 1.7% (11×),但**主信号是余量**(余量最小 20% 判错 26% vs 最大 20% 2.2%),σ 是自适应精修。
- **debias 已榨干**:FP/FN=1.69(偏过稳定),但 F1-最优平移仅 +0.008、收益 ~+0.005(还是 test 泄漏上界),按平均残差去偏(+0.0097)反而掉(0.8153→0.8123)。当前 4ep 模型已基本居中,**debias 不是杠杆**(历史 +0.08 是给严重欠训的早期模型)。
- **判错富集难家族 → 定向补数据是真杠杆**:判错结构元素富集 Pu 3.3× / F 1.9× / Te 1.7× / Br·Ge·Tl·Yb ~1.6×,即 **f-block / 卤化物 / 硫族**(吻合历史 worst-case)。→ 用 OMat/Alexandria 这些家族定向加训,把边界能量做准。
- **净结论**:F1 = 边界精度 × 难家族;LLPR 的价值是**瞄准**(补哪类、能量可信度),不是免费涨分。
